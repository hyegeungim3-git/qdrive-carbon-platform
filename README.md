# QDrive — AI 기반 탄소중립 운영 플랫폼 (프로토타입)

대구광역시 시내버스의 OBD·DTG 데이터를 기반으로 안전운전 → 연료 절감 → 탄소 감축을 관리하는 운영 플랫폼 UI 프로토타입입니다.

## 화면 구성

| 파일 | 내용 |
|---|---|
| `탄소중립 대시보드.dc.html` | 관제 웹앱 — 통합 운영(실시간 지도·히트맵), 탄소·연료, 안전운행 대시보드 + 운영관리(차량·운전자·노선) + **AI Planning**(전기버스 전환 시뮬레이터) + **Reports**(지자체 제출용 보고서) |
| `QDrive 기사앱.dc.html` | 8인치 태블릿(가로) 기사용 앱 — 내비형 실시간 코칭 + 운행 리포트 + 게이미피케이션(사내 주간 랭킹·배지) |
| `시민 탄소 대시보드.dc.html` | 시민 공개 페이지 — 감축 성과 카운터·소나무 환산·월별 추이·노선별 친환경 등급 조회 |
| `운수사 경영 대시보드.dc.html` | 운수회사 경영진용 — 월 손익 효과(연료비+예지정비−인센티브), 5개사 벤치마크, 기사 분포, 전환 검토, AI 경영 인사이트 |

네 화면이 4개 이해관계자를 커버합니다: **관제센터(운영) → 기사(현장 실행) → 운수회사 경영진(투자 판단) → 시민(공개 소통)**.

주요 인터랙션: 상단 기간 토글(일간/월간/연간 — 탄소·연료 성과와 추이 차트 연동), 알림 벨 드롭다운, 노선/기사 필터 chip, 차량번호 검색(결과 없음 빈 상태 포함), 전환 대수 슬라이더(감축량·투자·ROI 실시간 계산), 보고서 선택·미리보기, 시민 페이지 노선 등급 조회.

## 라이브 데모

**https://hyegeungim3-git.github.io/qdrive-carbon-platform/** — 랜딩에서 4개 화면으로 진입합니다. master 푸시마다 자동 갱신됩니다.

## 실행 방법

정적 파일이므로 빌드 없이 열립니다.

- 로컬: 저장소 루트에서 `python -m http.server` 실행 후 `http://localhost:8000/탄소중립 대시보드.dc.html` 접속 (`file://` 직접 열기보다 HTTP 권장)
- 공유: GitHub Pages 활성화 (Settings → Pages → Branch: main, / root) 후 발급 URL로 접속

지도 타일(CARTO)·차트(Chart.js)·지도 라이브러리(Leaflet)는 CDN에서 로드되므로 인터넷 연결이 필요합니다.

## 데이터 기준 (목업)

- 412대 (경유 272 · CNG 72 · 전기 68), 5개 운수회사
- 절감량 = 도입 전 12개월 연비 베이스라인 대비 실측 (에코드라이빙 실증 범위 3~7% 내)
- 배출계수: 경유 2.68 kgCO₂/L · CNG 2.22 kgCO₂/㎥ · 전력 0.4594 kgCO₂/kWh (환경부 고시)
